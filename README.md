# 軍儀（Gungi）プロジェクト

『HUNTER×HUNTER』に登場する架空のボードゲーム「軍儀」を実装し、深層強化学習によるAI対戦をWebブラウザで実現するプロジェクトです。

## 🎮 デモ

**オンラインでプレイ**: [https://gungi-game.onrender.com](https://gungi-game.onrender.com)

ブラウザで今すぐ軍儀をプレイできます！（Render上でホスティング）

## プロジェクト構成

```
Gungi/
├── src/
│   ├── engine/          # ゲームルールエンジン
│   │   ├── board.py     # 盤面管理
│   │   ├── piece.py     # 駒の定義と動き
│   │   ├── move.py      # 手の表現
│   │   └── rules.py     # ルール判定
│   ├── model/           # 深層学習モデル
│   │   ├── network.py   # ニューラルネットワーク
│   │   ├── mcts.py      # モンテカルロ木探索
│   │   └── trainer.py   # 学習ロジック
│   └── api/             # FastAPIサーバ
│       └── main.py      # APIエンドポイント
├── frontend/            # Web UI
│   ├── index.html       # メインページ
│   ├── game.js          # ゲームロジック
│   └── styles.css       # スタイル
├── tests/               # テストコード
└── requirements.txt     # Python依存パッケージ
```

## セットアップ

### 必要な環境

- Python 3.10以降
- Node.js 18以降（フロントエンド用）

### インストール

```bash
# Python依存パッケージ
pip install -r requirements.txt

# フロントエンド依存パッケージ（必要に応じて）
cd frontend
npm install
```

### 開発サーバの起動

```bash
# APIサーバ起動
python -m uvicorn src.api.main:app --reload --port 8000

# フロントエンド開発サーバ（別ターミナル）
cd frontend
npm run dev
```

## ゲームルール概要

- 9×9マスの盤面
- 各プレイヤー25枚の駒（14種類）
- 駒を最大3段まで積み重ね可能（ツケ）
- 相手の帥（すい）を取れば勝利

詳細は `ルール.txt` を参照してください。
